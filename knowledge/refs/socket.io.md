# socket.io

Socket.IO is a transport protocol that enables real-time bidirectional event-based communication between clients.

## В Jamming Bot

В [[jamming-bot/Jamming Bot|Jamming Bot]] с помощью `socket.io` соединяются Flask и веб-страницы для передачи событий и realtime-обновлений.
