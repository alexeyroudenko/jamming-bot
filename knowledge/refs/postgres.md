# postgres

Postgres — реляционная база данных для сервисов, которым нужно устойчивое структурированное хранение.

## В Jamming Bot

В [[jamming-bot/Jamming Bot|Jamming Bot]] Postgres используется в части сервисов как долговременное хранилище данных.
