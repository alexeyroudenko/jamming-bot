# Spacy

Spacy — это библиотека для обработки естественного языка на Python. Она используется для токенизации, лемматизации, выделения частей речи, именованных сущностей и синтаксического анализа.

## В Jamming Bot

В [[jamming-bot/Jamming Bot|Jamming Bot]] Spacy используется как основной NLP-слой для извлечения тегов, фраз, сущностей и семантических связей.
