# D3

**[D3.js](https://d3js.org/)** — это JavaScript-библиотека для визуализации данных с помощью HTML, SVG и CSS.

## В Jamming Bot

D3 используется как одна из референсных и потенциальных библиотек для data-driven визуализаций в [[jamming-bot/Jamming Bot|Jamming Bot]].
