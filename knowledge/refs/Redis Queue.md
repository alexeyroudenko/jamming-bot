# Redis Queue

Redis Queue (RQ) — библиотека для фоновых задач поверх Redis.

## В Jamming Bot

В [[jamming-bot/Jamming Bot|Jamming Bot]] RQ используется для jobs-пайплайна и фоновой обработки шагов, анализа и сохранения данных.
