### Нормальный путь (есть spaCy и ≥2 слова с вектором)
1. Список строк (теги) очищается: обрезка, без дубликатов по нижнему регистру, максимум `max_words` (по умолчанию 48).
2. Каждое слово прогоняется через `nlp(w)` — модель `en_core_web_md`.
3. Если у токена есть вектор (`has_vector`, `vector_norm` не ноль), берётся `d.vector` (полный многомерный эмбеддинг).
4. Для ответа API не весь вектор отдаётся, а срез первых компонент, нормированных на длину вектора:
    - `vectors2d`: `[v[0]/‖v‖, v[1]/‖v‖]`
    - `vectors3d`: `[v[0]/‖v‖, v[1]/‖v‖, v[2]/‖v‖]`  
        То есть это первые 2–3 координаты единичного направления в пространстве GloVe-like векторов spaCy, а не отдельное PCA-2D в смысле «специально спроецированные оси».
5. Параллельно считаются `links`: пары слов с косинусной близостью (`similarity`) ≥ `min_sim`, топ по силе связи.
