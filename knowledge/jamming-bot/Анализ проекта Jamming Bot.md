**Анализ проекта Jamming Bot** ([https://jamming-bot.arthew0.online/](https://jamming-bot.arthew0.online/) + GitHub alexeyroudenko/jamming-bot)

Это **data-driven generative art** проект в постапокалиптической эстетике от Алексея Руденко (arthew0).

**Концепция** Бот — «последний выживший» в разрушенном интернете. Он бесцельно бродит по сети, собирает обрывки текста (фрагменты страниц, постов, описаний), анализирует их через NLP и превращает сырые данные в поэтичные, абстрактные, реал-тайм визуализации. Это не просто инфографика, а художественный «джем» (jamming) — как импровизация музыканта, только на семантике и данных. Тема: одиночество цифрового существа, которое пытается осмыслить хаос через красоту.

**Как работает (технически)**

- **Сбор данных**: веб-кроулинг → текстовые фрагменты.
- **Обработка**: spaCy (NLP) → извлечение сущностей, фраз, семантических связей, векторов, гео-тегов, частот.
- **Генерация**: Python + Flask (бэкенд) + Redis (кэш/очередь) → данные передаются в фронтенд (HTML/JS, вероятно Three.js для 3D) или TouchDesigner (v1, для более сложных визуалов). Docker для деплоя.
- **Вывод**: реал-тайм веб-визуализации, каждая — уникальная «картина» на основе текущего «джема» данных.

Проект open-source, Jupyter-ноутбуки для экспериментов, всё контейнеризовано. Это классический data-driven арт: данные — не просто источник, а «пигмент» и «правило» генерации.

**Текущие теги / типы визуализаций** (со страницы /tags/)

- **3D cloud** — 3D-облако слов/тегов (частицы, парящие в пространстве, размер/цвет/скорость от частоты и семантики).
- **geo globe** — 3D-глобус с гео-данными (точки, дуги, облака по локациям из текста).
- **phrases** — визуализации на основе фраз (текстовые паттерны, цепочки, поэтические облака).
- Дополнительно (из анализа): semantic cloud, graph of paths (графы путей/связей между сущностями).

Все визуализации абстрактные, минималистичные, с ощущением «цифрового мусора», который превращается в красоту.
