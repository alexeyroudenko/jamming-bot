# Get tags

## URL

https://jamming-bot.arthew0.online/api/tags/get/

## Описание

Это служебная точка доступа к одному из главных художественных материалов проекта — тегам, полученным из текстовых фрагментов. Через этот API технический слой напрямую поддерживает визуальные композиции, где данные становятся формой и ритмом.
