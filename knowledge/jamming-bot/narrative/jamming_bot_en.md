# Jamming Bot EN

Jamming Bot is a fantasy about a post-apocalyptic future in which the main functions of the internet and assistant bots have collapsed, leaving only one self-replicating bot that aimlessly wanders through the remnants of the web. It is a bot that has no goal, only a path.

## Understanding

Currently, spiders, crawlers, and bots have a service purpose. They act as search engines, collect information, and automate internet infrastructure. In this project that logic survives after purpose itself has disappeared.

Jamming Bot symbolizes loneliness and persistence within technology that has moved beyond control and optimization. It behaves like an observer among fragments of information, trying to catch residual traces of humanity in scraps of text from ruined pages and forgotten systems.

## Path

The Jamming Bot algorithm resembles search engine robots: from each page it visits, it collects links to other pages and extracts text for further analysis. It is left to itself, without any regulatory system defining it. This touches on [[Экзистенциализм]], because the most surprising thing is that Jamming Bot is actually free.
