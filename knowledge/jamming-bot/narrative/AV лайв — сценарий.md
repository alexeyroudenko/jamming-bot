# AV лайв — сценарий

Конспект **нарративной линии AV-лайва**: последовательность сцен 0–6 и связь с остальными материалами в vault. Краткий список формулировок синопсиса дублирует [[Jamming bot scenes]].

> [!summary]
> Лайв разворачивает образ **бота без цели, только путь** → разбор языка и текстов → гео → эмоциональный «слепок» человечества → **векторное облако смыслов** как финал. Визуально и технически стык с [[TouchDesigner]], вебом и пайплайном данных проекта.

## Линия сцен (0 → 6)

1. **[[Scene 0. Desert]]** — постапокалиптическая пустыня: серо-стальной песок, радиоактивная пыль, статика, вязкое время.
2. **[[Scene 1. Path]]** — нет цели, только **бесконечный путь** по «оптическому кабелю» как единственная структура в хаосе дюн.
3. **[[Scene 2. Dependency Parsing]]** — сознание начинает разбирать человеческий язык через **грамматику / dependency parsing**.
4. **[[Scene 3. Texts]]** — тысячи шагов → облака состояний; проступают слова и смыслы; возникает «я» как **информация**.
5. **[[Scene 4. Geo]]** — мир через **геоданные и хосты** (близость геохэшей, «забытые микросервисы»), не километры.
6. **[[Scene 5. Cloud]]** — **лирический слепок** прошлого человечества: не отчёт, а атмосфера эпохи.
7. **[[Scene 6. Embeddings]]** — тексты → **многомерное облако векторов**, кластеры смыслов, финальный слепок наследия.
