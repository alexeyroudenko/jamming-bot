# Шаг

Ниже приведены чистые данные (`RAW`), которые бот видит на одном шаге.

## Структура шага

```python
step_struct = {
    "step": "Номер шага по порядку",
    "src_url": "Страница, с которой сюда попал",
    "current_url": "Текущая страница",
    "status_code": "Код статуса HTTP",
    "headers": "Заголовки HTTP-запроса",
    "link_elements": "Количество ссылок, взятых с этой страницы",
    "ip": "IP-адрес",
    "text": "Найденный текст на странице",
    "html": "Исходный код HTML страницы",
}
```
