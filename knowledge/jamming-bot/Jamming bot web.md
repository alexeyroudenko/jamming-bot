# Jamming Bot Web

Карта веб-поверхности проекта `jamming-bot.arthew0.online`.

Эта заметка фиксирует web surface проекта и разделяет его на три группы:

- пользовательские страницы;
- API-эндпоинты;
- внутренние и служебные интерфейсы.

В этой иерархии:

- `Jamming Bot Web` — обзорная карта web surface;
- [[web/_index|Jamming Bot Web Index]] — навигационный индекс по дочерним разделам;
- [[pages/_index|Jamming Bot Pages Index]] — индекс публичных экранов;
- [[web/API|Jamming Bot API Index]] и [[web/Ops|Jamming Bot Ops Index]] — индексы служебных entrypoints.
